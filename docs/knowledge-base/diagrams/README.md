# AI 客服与知识库方案图

本目录保存可维护的 Mermaid 图源文件及导出的 PNG 文件。

文件说明：

- `01-current-system-architecture.mmd`：当前项目整体结构图
- `02-current-ai-service-flow.mmd`：当前 AI 客服处理流程图
- `03-target-ai-service-flow.mmd`：优化后的 AI 客服目标流程图
- `04-rag-solution-comparison.mmd`：GraphRAG / LightRAG / llm-wiki 方案对比图
- `mermaid-config.json`：统一渲染样式配置

导出目标：

- 同名 `.png`

适用场景：

- 项目文档
- 汇报材料
- 后续技术方案讨论
